# Project Documentation

## Project Overview
This project aims to design an effective communication system that enhances interaction through modern technology. The goal is to create intuitive interfaces that facilitate seamless communication in various environments.

## Project Structure
- **/src**: Contains the main source code of the project.
- **/docs**: Comprehensive documentation of the project.
- **/assets**: Contains images, icons, and other assets used in the project.

## Technologies
- **Frontend**: React, Redux, CSS
- **Backend**: Node.js, Express
- **Database**: MongoDB

## Getting Started Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/tharinadacwa/Communication_Design_Project.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Communication_Design_Project
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the application:
   ```bash
   npm start
   ```

## Components
- **Header Component**: Responsible for the navigation bar.
- **Message Component**: Handles the display and sending of messages.
- **User Component**: Manages user interactions and settings.
